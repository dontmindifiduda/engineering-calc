Hazen Williams Equation:

$$H = {{10.44 L_{ft} Q_{gpm}^{1.852}} \over {C^{1.852} d_{in}^{4.8704}}}$$ where:

  • H = Head Loss (ft)
  • L = Length (ft)
  • Q = Flow (gpm)
  • C = Hazen Williams Coefficients
  • d = Pipe Diameter (in)
